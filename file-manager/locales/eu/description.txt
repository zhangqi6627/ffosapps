File Manager SD txartelaren fitxategiak ikusi eta kudeatzeko balio duen Firefox OS sistemarako aplikazio bat da.

Eskakizunak:

Aplikazio honek SD txartelara sartzeko baimena behar du.

Ezaugarriak:

- Fitxategiak kopiatzea, mugitzea, berizendatzea eta ezabatzea gaitzen du.
- Karpetak sortzea, berizendatzea eta ezabatzea gaitzen du.
- Musika, bideo, testu eta irudi fitxategiak irekitzen ditu.
- PDF fitxategiak irekitzen ditu (Firefox OS 1.4 bertsioa edo berriagoa behar du).
- Edozein musika, bideo, testu edo irudi fitxategi irekitzen ditu.
- Edozein irudi atzekalde bezala erabiltzeko gaitzen du.
- Edozein fitxategi bluetooth, email... bidez partekatu ditzake.
- Testu soileko fitxategiak sortu eta aldatzea gaitzen du.
- Edozein karpeta kudeatzea gaitzen du beraren gainean presio luze bat eginda.

Garatzaileentzako:

SD txartelarekin interakziorik dituen aplikazioren bat garatu baduzu, File Managerrek zure aplikazioa erabili dezake fitxategi horiek irekitzeko. Informazio gehiagorako bisitatu Githubeko orrialdea.

Baliabideak:

GPL-3.0+ lizentziapean dagoen kode irekiko aplikazio bat da.
Building Blocks liburutegiaren moldaketa bat eta Numix Circle ikono paketea erabiltzen ditu.

Aplikazio hau Firefox OS 1.1 duen ZTE Open gailu batean eta Firefox OS 1.3, 1.4 eta 2.0 dituen simulatzailean probatu da.
