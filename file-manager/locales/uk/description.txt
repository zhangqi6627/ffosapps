File Manager - це програма для Firefox OS, призначена для перегляду й керування файлами SD-карти.

Вимоги:

Ця програма потребує дозвіл на доступ до SD-карти.

Можливості:

- Дозволяє копіювати, переміщувати, перейменовувати й вилучати файли.
- Дозволяє створювати, перейменовувати й вилучати теки.
- Відкриває музичні файли, відео, текстові файли та зображення.
- Відкриває файли PDF (необхідна Firefox OS 1.4 або вище).
- Відкриває будь-який файл як музику, відео, текст чи зображення.
- Дозволяє встановлювати шпалерами будь-яке зображення.
- Дозволяє ділитися будь-якими файлами через bluetooth, електронну пошту та інше.
- Дозволяє створювати й редагувати текстові файли.
- Дозволяє керувати будь-якою текою, довгим натисканням на ній.

Перекладіть цю програму своєю мовою:

Ви можете допомогти з перекладом File Manager своєю мовою, просто створіть fork репозиторію або завантажте цей файл, перекладіть його та надішліть мені електронною поштою. Нові мови будуть додаватися кожні вихідні, тож будьте терплячими.

Для розробників:

Якщо ви розробили програму, яка взаємодіє з файлами на SD-карті, File Manager може використовувати вашу програму для безпосереднього відкриття файлів. Для отримання докладнішої інформації, перегляньте репозиторій на Github.

Джерела:

Це програма з відкритим програмним кодом на умовах ліцензії GPL-3.0+.
Використовує ваніант бібліотеки Building Blocks та піктограми від Numix Circle.

Ця програма пройшла тестування на пристрої ZTE Open з Firefox OS 1.1, а також симуляторах для Firefox OS 1.3, 1.4 і 2.0.
