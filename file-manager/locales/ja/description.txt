File Managerは、SDカードからファイルを探索し管理するためのFirefox OS用アプリです。

要件:

このアプリは、SDカードへのアクセス許可が必要です。

特徴:

- ファイルのコピー、移動、名前の変更、削除ができます。
- フォルダの作成、名前の変更、削除ができます。
- 音楽、動画、テキスト、画像など任意のファイルが開けます。
- PDFファイルが開けます（Firefox OS 1.4以降が必要）。
- 画像を壁紙に設定することができます。
- Bluetooth、電子メールなどでファイルを共有することができます。
- テキストファイルの作成、編集ができます。
- フォルダをロングタッチすると、そのフォルダの管理が行えます。

開発者の方へ:

SDカード内のファイルを閲覧・再生・実行するアプリを開発されている場合は、File Managerがファイルを直接そのアプリで開くよう設定することができます。詳細はGitHubのリポジトリを参照してください。

リソース:

このアプリは、GPL-3.0+に基づくオープンソースソフトウェアです。
Building BlocksライブラリとNumix Circleアイコンテーマを使用しています。

このアプリは、Firefox OS 1.1を搭載したZTE Open端末とFirefox OS 1.3、1.4、2.0のシミュレータで動作確認をしました。
